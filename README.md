### Resources Consulted:
Gemini and Claude were used as code assistants. We worked together on the app using discord as our discussion tool. All other resources utilized were prior submitted project 3 materials, such as reports and low/medium fidelity prototypes that we iterated off of.

### Usage:
The entire project is a monolithic HTML. You can simply open the file in the web browser of your choice and use the site. The site was intended to be used on desktop, with the phone simulating an app, however 99% of features work perfectly fine on mobile as well, barring the word map interactivity sometimes producing unexpected results.
